# 🚲 Bike Sales Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Bike Sales Dashboard built entirely in Microsoft Excel. The dashboard provides insights into customer demographics and purchasing behavior using Pivot Tables, Pivot Charts, Slicers, and Excel formulas.

The goal is to help identify the factors that influence bike purchases and allow users to interactively filter the data.

---

## 📂 Dataset

The dataset contains customer information including:

- Customer ID
- Marital Status
- Gender
- Income
- Number of Children
- Education
- Occupation
- Home Ownership
- Number of Cars
- Commute Distance
- Region
- Age
- Bike Purchase Status

Additionally, an **Age Bracket** column was created to group customers into meaningful age categories.

## 📊 Dashboard Overview

The dashboard includes:

### 💰 Average Income Analysis
- Compares average income by:
  - Gender
  - Bike Purchase Status

### 👥 Customer Age Brackets
- Analyzes bike purchases across different age groups.
- Categories:
  - Adolescent
  - Middle Age
  - Old

### 🚗 Customer Commute Analysis
- Shows the relationship between commute distance and bike purchases.

### 🎛 Interactive Filters (Slicers)

Users can filter the dashboard using:

- Marital Status
- Region
- Education
- Gender

All charts update dynamically based on slicer selections.

---

## 📈 Key Insights

- Male customers who purchased bikes had the highest average income.
- Middle-aged customers represented the largest customer segment.
- Commute distance appears to influence purchasing behavior.
- The dashboard allows quick comparisons across regions, education levels, and customer demographics.
